4) Copiamos  el archivo de configuración y configuramos /etc/network/interfaces

#La interfaz para host-only
auto enp0s8
iface enp0s8 inet static
address 192.168.120.100
netmask 255.255.255.0
gateway 192.168.120.1 
Systemctl restart networking.service
Si falla se modifica el ICMP del cortafuegos de Windows
Aunque salga un error, utilizando ip addr se ha añadido correctamente y funciona el servicio. O ifconfig
